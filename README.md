# DeepLearningIndaba2026-Scientific-Narrative-Academy
Official repository for the Deep Learning Indaba 2026 Skill Session: Mastering the Scientific Narrative – Reading and Writing High-Impact AI Research Papers.

## Repository Structure

Every link below goes straight to the real folder or file in this repository.

- [`docs/`](docs/) — session abstract, session plan, participant handbooks, deliverables checklist
- [`slides/`](slides/) — the deck (40 slides): [`PPTX/`](slides/PPTX/) (editable) · [`PDF/`](slides/PDF/) (print-ready) · `images/` (pending)
- [`resource-kit/`](resource-kit/) — the participant resource kit:
  - [`checklist/`](resource-kit/checklist/) — Scientific Writing Checklist (PDF)
  - [`exercises/`](resource-kit/exercises/) — 3-sentence abstract exercise + scenario cards
  - [`reading-list/`](resource-kit/reading-list/) — annotated reading list + `references.bib`
  - [`overleaf/`](resource-kit/overleaf/) — the guided, in-session LaTeX exercise + rendered preview
  - [`offline-pack/`](resource-kit/offline-pack/) — printable backup for no-power/no-Wi-Fi rooms
  - [`demo-paper/`](resource-kit/demo-paper/) · [`reference-managers/`](resource-kit/reference-managers/) — pending
- [`templates/`](templates/) — venue-agnostic paper starters:
  - [`overleaf/`](templates/overleaf/) — LaTeX starter + [`bonus-figures/`](templates/overleaf/bonus-figures/) example plots
  - [`neurips/`](templates/neurips/) · [`iclr/`](templates/iclr/) · [`ieee/`](templates/ieee/) — pending
- [`examples/`](examples/) — good/weak abstract & introduction samples (pending)
- [`references/`](references/) — [`bibliography.bib`](references/bibliography.bib) + [`papers/`](references/papers/) · [`books/`](references/books/) · [`guides/`](references/guides/)
- [`assets/`](assets/) — QR code, logo, banner, [`Photos/`](assets/Photos/) (pending)
- [`website/`](website/) — minimal GitHub Pages landing page
- [`zenodo/`](zenodo/) — archival metadata for a future DOI

Root files: [`LICENSE`](LICENSE) · [`CITATION.cff`](CITATION.cff) ·
[`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) · [`CONTRIBUTING.md`](CONTRIBUTING.md)

Folders marked "pending" are scaffolded with a `README.md` describing what
belongs there but don't have real content yet — see
[CONTRIBUTING.md](CONTRIBUTING.md) if you'd like to help fill them in.
