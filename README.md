## 🎯 About this Session

To achieve **Sovereign Intelligence**, African researchers must not only build innovative models but also master the art of global scientific communication. This 90-minute, hands-on Skill Session provides a systematic, reusable methodology for **consuming** dense AI literature (Keshav's 3-Pass Reading Method) and **producing** publication-grade research (the Scientific Writing Blueprint), with worked examples anchored in African AI use cases — climate, agriculture, health, and low-resource languages.

**Facilitator:** Dr. Ing. Castro Gbêmêmali Hounmenou — University of Labé (UL) / CERFIG, Guinea.

**Audience:** Undergraduate, Master's, and early-PhD students in AI/ML. No prior publishing experience required.

## 📚 Learning Outcomes

By the end of this session, participants will be able to:

1. **Triage and deconstruct** any NeurIPS/ICLR-style paper using Keshav's 3-Pass Method (scan → grasp → reconstruct).
2. **Engineer a 3-sentence abstract** using the Context+Gap → Method → Results+Impact formula.
3. **Structure a full paper** with the four-step Writing Blueprint: Frame, Core, Impact, Honesty.
4. **Design honest, publication-quality figures** and apply FAIR data principles.
5. **Position African-led research** for high-impact international venues while retaining local relevance.

## 🚀 Quick Start

This is a documents-and-templates repository — there is no code to install.

```bash
git clone https://github.com/Castro2026/DeepLearningIndaba2026-Scientific-Narrative-Academy.git
```

- **Read the deck:** [`slides/PDF/Livrable3_Slides_Scientific_Narrative_v3.pdf`](slides/PDF/Livrable3_Slides_Scientific_Narrative_v3.pdf)
- **Do the exercise:** upload [`resource-kit/overleaf/indaba-narrative-template.zip`](resource-kit/overleaf/indaba-narrative-template.zip) to [Overleaf](https://www.overleaf.com) (New Project → Upload Project)
- **Start your own paper:** use the venue-agnostic starter in [`templates/overleaf/`](templates/overleaf/)
- **Follow the checklist:** [`resource-kit/checklist/Scientific_Writing_Checklist.pdf`](resource-kit/checklist/Scientific_Writing_Checklist.pdf)

## 🌍 Sovereign Intelligence Framing

Mastering research communication is an act of scientific self-determination.
As the Participant Handbook puts it: *"Sovereign Intelligence is not
isolation; it is authorship. It means African researchers defining the
problems, collecting the data under African governance, publishing the
results under African names, and being cited as the origin — while engaging
the global community as peers."*
