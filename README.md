# DeepLearningIndaba2026-Scientific-Narrative-Academy
Official repository for the Deep Learning Indaba 2026 Skill Session: Mastering the Scientific Narrative – Reading and Writing High-Impact AI Research Papers.

## Repository Structure

```
.
├── README.md
├── LICENSE
├── CITATION.cff
├── CODE_OF_CONDUCT.md
├── CONTRIBUTING.md
├── .gitignore
│
├── docs/                      # session abstract, plan, participant handbook
├── slides/
│   ├── PPTX/ · PDF/           # the deck (40 slides)
│   └── images/                # figures/ · icons/ · diagrams/ (pending)
│
├── resource-kit/
│   ├── checklist/             # Scientific Writing Checklist (pending)
│   ├── exercises/             # 3-sentence abstract exercise + scenario cards
│   ├── reading-list/          # references.bib — the 6 papers behind the session
│   ├── demo-paper/            # live Pass-1 demo excerpt (pending)
│   ├── reference-managers/    # Zotero / Mendeley guides (pending)
│   ├── overleaf/              # the guided LaTeX template + preview
│   └── offline-pack/          # printable backup for no-power/no-Wi-Fi rooms
│
├── templates/                 # generic venue starters: neurips/ iclr/ ieee/ overleaf/ (pending)
├── examples/                  # good/weak abstract & introduction samples (pending)
├── references/                # bibliography.bib + papers/ books/ guides/
├── assets/                    # QR code, logo, banner, photos/ (pending)
├── website/                   # minimal GitHub Pages landing page
└── zenodo/                    # archival metadata for a future DOI
```

Every folder above exists in the repository today. Folders marked "pending"
are scaffolded with a `README.md` describing what belongs there but don't
have real content yet — see [CONTRIBUTING.md](CONTRIBUTING.md) if you'd like
to help fill them in.
