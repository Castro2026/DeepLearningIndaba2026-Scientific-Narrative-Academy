# offline-pack/

A self-contained backup for rooms with no power, projector, or Wi-Fi (see the
"Contingency Plan" in `docs/Detailed_Plan_Final.docx`): printed handouts of
all slides and exercise sheets, plus the repository URL as a fallback for the
QR code.
