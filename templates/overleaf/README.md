# LaTeX / Overleaf template

**Fastest start:** create a free account on [Overleaf](https://www.overleaf.com),
click **New Project → Upload Project**, and upload this folder as a `.zip`
(`main.tex` + `references.bib`). It compiles out of the box with pdfLaTeX.

**For a real submission**, always download the *official* style files of the
venue and paste your sections into them:

| Venue | Where to get the official template |
|---|---|
| NeurIPS | neurips.cc → *Paper Information / Style files* |
| ICLR | iclr.cc → *Call for Papers* (author guide) |
| ACL / AfricaNLP | github.com/acl-org/acl-style-files |
| Overleaf gallery | Menu → *Templates* → search the venue name |

Every `%` comment in `main.tex` is a coaching note tied to the Writing
Blueprint (Frame · Core · Impact · Honesty) and to the Checklist in
[`resource-kit/checklist/`](../../resource-kit/checklist/). Delete the
comments as you replace them with your own text.

See also [`bonus-figures/`](bonus-figures/) for example result figures
(Pareto, box plot, confusion matrix, convergence curves) illustrating the
"honest, labeled figures" step of the Blueprint, and
[`resource-kit/overleaf/`](../../resource-kit/overleaf/) for the workshop's
own guided, in-session abstract-writing exercise (with coaching boxes).
